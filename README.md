# Teste Vaga Front End

Projeto desenvolvido para o processo seletivo na Econverse onde temos uma página desenvolvida em React e Typescript.

## Instalação

### Clone o repositório
```
git clone https://github.com/Thiaki/teste-front-end.git
```

### Entre no diretório do projeto
```
cd teste-front-end-jr
```

### Instale as dependências
```
yarn install
```
### Uso
Para rodar o projeto localmente, execute o comando:
```
yarn dev
```
Isso iniciará o servidor de desenvolvimento. Abra http://localhost:5173 no seu navegador para visualizar o projeto.

## Tecnologias Utilizadas
Principais tecnologias e ferramentas utilizadas no projeto:

- React
- TypeScript
- Sass
